# Roadmap

The roadmap prioritizes reusable workflows over repository size.

## Phase 1 — Foundation

- Define contribution and recipe standards
- Publish the feature-planning skill
- Add prompt and skill templates
- Add pre-build and pre-launch checklists
- Document the expected repository structure

## Phase 2 — Core product workflows

- UX review skill
- Bug investigation skill
- Data-model planning skill
- Authentication and permissions checklist
- Supabase integration guide
- Deployment readiness checklist

## Phase 3 — Production quality

- Security review workflow
- Accessibility audit workflow
- Performance review workflow
- Analytics and event-tracking plan
- SEO review workflow
- Testing strategy templates

## Phase 4 — Example applications

- SaaS dashboard
- Marketplace
- Content platform
- Internal operations tool
- Booking application

## Non-goals

- A dump of untested prompts
- Claims that one prompt works for every project
- Copying proprietary documentation
- Replacing product judgment, code review, or security testing
