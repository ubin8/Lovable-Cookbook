# Pre-build checklist

- [ ] The target user and problem are explicit.
- [ ] The expected outcome is measurable or observable.
- [ ] MVP scope is separated from later ideas.
- [ ] Non-goals are documented.
- [ ] Main user flows and error paths are understood.
- [ ] Data entities and ownership are identified.
- [ ] Authentication and permission rules are defined.
- [ ] External integrations and failure modes are known.
- [ ] Sensitive data and security implications are reviewed.
- [ ] Acceptance criteria are testable.
